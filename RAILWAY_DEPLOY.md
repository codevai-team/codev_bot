# Деплой на Railway.app (БЕСПЛАТНО)

## Почему Railway?
- ✅ Бесплатный план ($5 кредитов в месяц)
- ✅ Автоматическое определение Python
- ✅ Простой интерфейс
- ✅ Встроенная PostgreSQL база

## Шаги для деплоя:

### 1. Создайте аккаунт
Перейдите на https://railway.app и войдите через GitHub

### 2. Создайте новый проект
1. Нажмите "New Project"
2. Выберите "Deploy from GitHub repo"
3. Выберите репозиторий `codevai-team/codev_bot`

### 3. Добавьте PostgreSQL базу
1. В проекте нажмите "New" → "Database" → "Add PostgreSQL"
2. Railway автоматически создаст переменную `DATABASE_URL`

### 4. Добавьте переменные окружения
В настройках сервиса ("Variables") добавьте:

**Обязательные:**
- `BOT_TOKEN` - токен Telegram бота (от @BotFather)

**Опциональные:**
- `IMGBB_API_KEY` - для загрузки изображений

**Примечание:** `DATABASE_URL` уже создана автоматически при добавлении PostgreSQL

### 5. Деплой
Railway автоматически:
1. Обнаружит Python
2. Установит зависимости из `requirements.txt`
3. Запустит `python main.py`

### 6. Инициализация базы данных
После первого деплоя:
1. Откройте "PostgreSQL" → "Data" → "Query"
2. Выполните SQL для создания таблиц
3. Или используйте скрипт инициализации

### 7. Добавьте администратора
Используйте скрипт `add_me_as_admin.py` или `init_admin.py`

## Мониторинг
- Логи: вкладка "Deployments" → выберите деплой → "View Logs"
- Метрики: вкладка "Metrics"

## Автоматический деплой
Railway автоматически деплоит при каждом push в GitHub
